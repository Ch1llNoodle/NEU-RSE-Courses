# 机器人基础原理
这门课程坐标变换与运动学建模比较难，最后老师还会考一些概念。概念部分请结合mooc习题进行学习（2019年好多填空题是原题），对于建模部分强烈推荐[公众号:ROBOTICS](https://mp.weixin.qq.com/s/jx2ViAcofXmrgsbLAW2_3Q),其中干货部分文章讲解十分透彻。  
实验内容仅供参考。
